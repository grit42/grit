# Using Assay Models

A list of all _Published_ Assay Models is available under the **Assay Models** tab. Clicking a row opens the Assay Model details page, which includes the list of experiments, data sheet definitions, metadata definitions, and results for all published experiments.

![Assay models](./assets/assay_models.png)

## Assay Model details

Opening an Assay Model reveals general information about the model and the following tabs:

### Experiments

Lists all experiments associated with this model. New experiments can be created directly from this tab using the **New** button above the table.

![Assay model](./assets/assay_model.png)

### Data

Shows data across all data sheets and experiments. Displayed experiments can be filtered by metadata values.

![Assay model data](./assets/assay_model_data.png)

### Data Sheet Definitions

Lists the data sheet definitions and their columns for reference.

![Assay model data sheets](./assets/assay_model_data_sheets.png)

### Metadata Definitions

Lists the metadata definitions for reference.

![Assay model metadata](./assets/assay_model_metadata.png)
