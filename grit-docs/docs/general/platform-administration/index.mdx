# Platform administration

The grit platform can be managed by one or more users having the _Administrator_ role. The default **admin** user has that role by default and can be used to administer the platform.

Administrators manage users and their authorisations, and supporting entities such as _Origins_ and _Locations_.

:::info
The _Administrator_ role is required to access the **Administration** section.
:::

:::caution
Users with the _Administrator_ role have no restrictions. Assign this role with caution.
:::
