# Metadata Templates

Metadata Templates enable setting many Metadata Values at once when creating Experiments, by providing groups of values that are often used together.

## Creating Metadata Templates

To create Metadata Templates, navigate to the Assay Model administration, then:

1) Select the *Metadata templates* tab
2) Click the New button

![Metadata templates](./assets/metadata_templates.png)

1) Provide a name and an optional description
2) Select values for some or all **Metadata Definitions**
4) Click **Save**

![Edit metadata templates](./assets/edit_metadata_templates.png)

## Modifying Metadata Templates

To modify an existing Metadata Template, click it in the list, make your changes and click **Save**.
