# Data sheets

**Data Sheets** describe the shape of the experimental data sets expected by this **Assay Model**.

Each **Data Sheet** represent an individual table in your data set, along with its columns, their names, data types and constraints. They must be defined once, and are used by all **Experiments** of this **Assay Model**.
