# Assay Models

**Assay Models** define structures for **Experiments** through **Data Sheets** and **Assay Model Metadata**.

**Data Sheets** are blueprints for tables storing experimental data, providing information about their columns.

**Assay Model Metadata** set **Metadata Definitions** for which **Experiments** must provide values.
