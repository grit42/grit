# Managing Assay Model Metadata

**Assay Model Metadata** configure which **Metadata Definitions** must have a value in **Experiments** belonging to this **Assay Model**.

Configured **Metadata Definitions** of an Assay Model can be viewed and modified under the **Metadata** tab.

![Assay Model Metadata](./assets/assay_model_metadata.png)

## Adding and Removing Assay Model Metadata

To add or remove Assay Model Metadata, click the **Edit** button above the table, and

![Edit Assay Model Metadata](./assets/edit_assay_model_metadata.png)

1) Click a Metadata Definition in the **Available** list to add it to the **Assay Model**
2) Click a Metadata Definition in the **Selected** list to remove it from the **Assay Model**
3) Click **Done** to exit
