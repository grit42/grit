# Managing Assay Models

:::note
Managing Assay Models requires the **Administrator** or **AssayAdministrator** role.
:::

Assay Models can be managed in the Assay Model administration section, which can be accessed as follows:
1) Go to the Assay Models menu
2) Click the gears icon top right in the toolbar

![Assay Models administration](../../assets/assay_models_administration.png)

## Creating an Assay Model

To create an Assay Model, navigate to the Assay Model administration, then:

1) Select the *Assay Models* tab
2) Click the New button

![Manage Assay Models](./assets/manage_assay_models.png)

1) Provide a name and an optional description
3) Select the appropriate **Assay Type**
4) Click **Save**

![Create Assay Model](./assets/create_assay_model.png)

## Editing an Assay Model

:::note
Published Assay Models cannot be modified.
:::

To modify an existing Assay Model, click it in the list, select the **Details** tab, make your changes, and click **Save**.

![Edit Assay Model](./assets/edit_assay_model.png)
